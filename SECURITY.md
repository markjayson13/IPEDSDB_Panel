# Security

IPEDSDB_Panel uses public IPEDS source data and should not contain private student records, institutional credentials, secrets, or local tokens.

Report security concerns by opening a private GitHub security advisory for the repository. If that is unavailable, use the current contact information at `https://markjayson.com`.

Do not open a public issue for:

- leaked credentials
- unpublished local paths that expose a private system
- dependency behavior that allows arbitrary file writes or command execution
- data files that appear to contain private or restricted records

For normal data-quality problems, use the issue templates in `.github/ISSUE_TEMPLATE/`.
